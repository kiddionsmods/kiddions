## 🛠️ Troubleshooting Common Issues

❌ **Overlay Does Not Appear**
The menu launches but nothing shows in-game. This usually means GTA 5 loaded with a different DirectX version than expected.
**Solution:** Add `-DX11` to your GTA 5 launch options in Steam or Epic, then restart both the game and Kiddions Mod Menu.

❌ **Antivirus Quarantines the Executable**
Windows Defender or third-party AV moves `Kiddions_Menu.exe` to quarantine immediately after extraction.
**Solution:** Add the full extraction folder path (for example `C:\Tools\KiddionsMM`) to your antivirus exclusion list before extracting the archive.

❌ **Menu Crashes on Launch**
The `.exe` opens briefly and then closes without an error message.
**Solution:** Ensure you are running Kiddions Mod Menu as Administrator and that your .NET runtime is version 4.7.2 or higher. Check `kiddions_log.txt` for details.

❌ **Features Stop Working After a GTA 5 Update**
A new game patch changes internal memory offsets, causing features like money loop or teleportation to fail silently.
**Solution:** Wait for the Kiddions Mod Menu team to release a compatibility patch. Never force an outdated version onto a new game build.

❌ **Low FPS While Overlay Is Active**
Performance dips when the overlay is visible, especially on older hardware.
**Solution:** Set `overlay.opacity` to `0.6` or lower in `config.json`, and disable `show_fps` if you do not need it.
